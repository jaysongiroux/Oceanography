Introduction

This tool will be used to assist oceanography researchers by giving researchers an easy, yet powerful, platform to enter data and output an intuitive graphs and analysis for professional documentation and research papers.

Tools

Wave Classification Tool
Features

This tool allows users to record just the Depth, Period and Wave Length and it will return the classification, Angular Frequency, Wave number and wave velocity
